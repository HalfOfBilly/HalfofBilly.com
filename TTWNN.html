Welcome to the certified-unofficial archive of The Town With No Name animation and image assets!
So far, this archive has most of the game's image and animation files in .gif format. There are likely still missing assets because of the nature of the files (unformatted file names).
If you find another image/animation I've missed, or have general feedback/corrections regarding the archive, feel free to Email me (link provided in home page)

The game's assets are kept in town.bin, which can be extracted using tools such as AnyBurn
From there, the extracted files don't have any extension. The picture and the animations are in .IFF format, converted into .GIF using IFF2GIF on GitHub.

DOWNLOAD ALL ASSETS AS .ZIP
NOTICE: Please always proceed with caution when downloading files from the internet. Although .zip files are great for transferring multiple files online, they can also be used malicioudly to bypass Malware through antivirus checks.
The linked .zip file is not malicious- it is only a collection of .gif. Regardless, you should also proceed with caution, and scan extracted files with an antivirus program.
If you have any concerns about this file, you may download the open source version here on GitHub
